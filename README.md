# EDA_spotify
Spotify dataset analysis using Python, Pandas, and Seaborn. Explores audio features, popularity trends, and applies clustering to group songs with similar characteristics.

## Dataset Description
The dataset contains audio statistics, such as "loduness", "tempo" etc. on the top 2000 tracks on Spotify from 2000-2019.

## Goals 
- Gain an overview of audio features and metadata
- Identify patterns and trends in time series
- Find relationships between features
- Group songs based on their audio features

## Key Steps
- Data Importing 
- Checking data quality 
- Data Cleansing
- Univariate Analysis and Outliers
- Bivariate Analysis
- Categorial Analysis
- Development of Features over time
- Summary of EDA
- Preparations for Clustering
- Clustering

## Insights & Findings
- There is a slight correlation between energy and loudness
- pop music has the most songs in this data set
- hip hop music is on average the most danceable genre
- Tempo on songs increased over the years
- Clustering reveals distinct song groups

## Methods
- Exploratory Data Analysis (EDA)
- K-Means Clustering
